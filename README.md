# IncaTern-Tutorial
This project demonstrates how to implement Firebase Authentication in a Flutter application using email and password login. The app connects a Flutter project to Firebase and shows how to build basic authentication features such as user sign-up, login, and handling authentication state.

The project uses the Flutter packages firebase_core and firebase_auth to integrate Firebase services and securely manage user authentication.

Features -
User sign-up with email and password
User login authentication
Firebase project configuration using FlutterFire
Basic authentication state handling
